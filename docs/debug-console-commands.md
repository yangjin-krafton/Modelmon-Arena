# Modelmon Arena — 콘솔 디버그 명령

## 개요

브라우저 개발자 도구 콘솔에서 `__mdm.*` 네임스페이스로 디버그 명령을 실행한다.

앱 로드 시 콘솔에 명령 목록이 자동 출력된다.
언제든지 `__mdm.help()` 로 다시 확인할 수 있다.

## 코드 위치

| 역할 | 파일 |
| --- | --- |
| 명령 등록 · 도움말 출력 | `src/system/core/debug.js` |
| 전투 debug 함수 구현 | `src/system/ui/battle.js` (export) |
| 초기화 호출 | `src/system/main.js` → `initDebug()` |

새 명령 추가는 `debug.js`의 `COMMANDS` 배열에 항목 하나를 추가하면 된다.

---

## 명령 목록

### `__mdm.win()`

현재 전투를 즉시 승리 처리한다.

- 적 HP를 0으로 설정하고 승리 결과 화면을 표시한다.
- 전투 중(`choosing` 또는 `animating` phase)에만 유효하다.

```js
__mdm.win()
```

---

### `__mdm.lose()`

현재 전투를 즉시 패배 처리한다.

- 플레이어 HP를 0으로 설정하고 셧다운 결과 화면을 표시한다.
- 전투 중에만 유효하다.

```js
__mdm.lose()
```

---

### `__mdm.skip()`

현재 대기 중인 다이얼로그를 모두 건너뛰고 스킬 선택 단계로 이동한다.

- `animating` phase에서만 유효하다.
- 긴 배틀 다이얼로그 텍스트를 빠르게 넘길 때 사용한다.

```js
__mdm.skip()
```

---

### `__mdm.hp(side, value)`

특정 몬의 HP를 강제 설정한다.

| 파라미터 | 타입 | 값 |
| --- | --- | --- |
| `side` | string | `"player"` 또는 `"enemy"` |
| `value` | number | 설정할 HP (0 ~ maxHp 범위로 클램핑) |

```js
__mdm.hp('enemy', 1)    // 적 HP를 1로 설정 (다음 공격에 쓰러짐)
__mdm.hp('player', 999) // 플레이어 HP를 maxHp까지 회복
```

---

### `__mdm.state()`

현재 전투 상태를 콘솔 테이블로 출력한다.

출력 내용: `phase`, `turn`, 양측 `HP / maxHp`

```js
__mdm.state()
```

---

### `__mdm.start(monId)`

지정한 몬 ID로 전투를 즉시 시작한다. 스타터 선택 화면을 건너뛴다.

- `monId`는 3자리 문자열 또는 숫자 모두 허용 (`"4"` → `"004"` 로 패딩).
- 상대방은 나머지 기본 스타터 중 랜덤 선택된다.

```js
__mdm.start('001')  // GPT-3.5 로 전투 시작
__mdm.start('004')  // Claude Haiku 로 전투 시작
__mdm.start(7)      // 자동 패딩 → "007"
```

---

### `__mdm.reset()`

localStorage 세이브 데이터를 초기화하고 페이지를 새로고침한다.

- 실행 전 확인 다이얼로그가 표시된다.
- 포획 기록, 스타터 해금 목록이 모두 초기화된다.

```js
__mdm.reset()
```

---

### `__mdm.help()`

명령 목록을 콘솔에 다시 출력한다.

```js
__mdm.help()
```

---

## 일반 디버그 시나리오

### 전투 빠르게 검증하기

```js
// 1. 인게임 탭에서 스타터 선택 후 전투 시작
// 2. 다이얼로그 스킵
__mdm.skip()
// 3. 적 HP 1로 설정
__mdm.hp('enemy', 1)
// 4. 아무 스킬 선택 → 자동 승리
```

### 특정 몬으로 바로 테스트

```js
// 스타터 선택 화면 없이 Claude Haiku 로 전투 시작
__mdm.start('004')
```

### 승리/패배 결과 화면 확인

```js
__mdm.win()   // 승리 결과 화면
__mdm.lose()  // 셧다운 결과 화면
```

### 세이브 초기화 후 처음부터

```js
__mdm.reset() // 확인 후 새로고침
```
