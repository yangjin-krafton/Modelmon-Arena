
# 모델몬 1기 스킬 도감

포켓몬 1기 기술 165개 전체를 기준으로 만든 `스킬 전용 도감` 샘플 폴더입니다.

이번 버전은 기술 목록을 단순 재해석하는 수준을 넘어서, `모델몬 전투용 원소 12개`와 `상태군 체계`를 기준으로 다시 정리한 상태입니다.

구성:

- `data/modelmon-skill-dex-gen1-full.csv`: 1기 전체 스킬 매핑 CSV
- `index.html`: 스킬 도감 화면
- `app.js`: CSV 파싱과 필터 로직
- `styles.css`: 화면 스타일

데이터 구조:

- 원본 기술명
- 원본 타입 / 분류 / 위력 / 명중 / PP
- 원본 효과 메모
- 전투 원소
- 능력 패턴
- AI 개념
- AI 키워드
- 우리 게임용 효과 설명
- 매핑 메모
- 상태군
- 상태 원소

실행:

```bash
cd sandbox/modelmon-skill-dex-gen1-full
python3 -m http.server 8092
```

또는 서버 없이 `index.html`을 직접 열어도 됩니다.

메모:

- `file://` 실행을 위해 CSV 내용을 `index.html` 안에도 같이 넣어 두었습니다.
- 이 폴더는 `개별 모델몬 스킬셋`이 아니라 `1기 전체 스킬 라이브러리`를 보는 용도입니다.
- 목록은 `스킬명 + 전투 원소 + AI 개념 + 상태군` 중심으로 보고, 상세에서는 `매핑 메모 + 상태 배치`까지 확인합니다.
